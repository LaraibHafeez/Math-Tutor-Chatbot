# Math Tutor Chatbot 🤖
An AI-powered math tutor chatbot built with Python and OpenAI API.
Solves math problems step by step in simple English.
## Features
- Solves algebra, geometry, calculus problems
- Explains step-by-step solution
- Simple conversational interface
## Technologies Used
- Python 3
- OpenAI API (GPT-3.5-turbo)
## How to Run
1. Clone the repository
   git clone https://github.com/LaraibHafeez/Math-Tutor-Chatbot.git
2. Install dependencies
   pip install openai
3. Set your API key
   set OPENAI_API_KEY=your_api_key_here
4. Run the chatbot
   python chatbot.py
## Example
You: Solve 2x + 5 = 15
Tutor: Step 1: Subtract 5 from both sides → 2x = 10
       Step 2: Divide both sides by 2 → x = 5
